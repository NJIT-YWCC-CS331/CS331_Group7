# CS331_Group7
Web app for Car Rental
Template by phpjabbers.com 

https://web.njit.edu/~zhm5/CS331Project/index.php

Allows Users to
1) create and login into an account
2) view profile information
3) search for a car rental
4) book a car rental

Allows Admins to
1) view registered users
2) view reservations and template
